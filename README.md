<h1>Quick Chat App</h1>

<h3> Features of the Chat App  </h3>
<ul> 
<li>User can create an account and sign in </li>
<li> User can create his chat account, share the chat id to the person he wants to chat with and chat with the person </li>
<li> User can Send Giphys and Images in the Chat </li>
<li> The Chat Name and Chat Profile Picture can be updated </li>
<li> There is track of who is entering the chat and who makes changes to the chat profile</li>
</ul>
